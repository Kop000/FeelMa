🔎Python 标准库 — Python 3.12.0 文档

快速入门

  1. 当语句以冒号:结尾时，缩进的语句视为代码块。

  2. Python允许在数字中间以_分隔，因此，写成10_000_000_000和10000000000是完全一样的。十六进制数也可以写成0xa1b2_c3d4。

  3. 字符串是以单引号'或双引号"括起来的任意文本，用'''...'''的格式表示多行内容。

  4. 布尔值：True、False；逻辑运算：and、or、not

  5. 空值是Python里一个特殊的值，用None表示。None不能理解为0，因为0是有意义的，而None是一个特殊的空值，在Python的交互环境不显示结果，相当于null，return None可以简写为return，和java里一样。

  6. Python是动态语言，Java是静态语言。同一个变量可以反复赋值，而且可以是不同类型的变量

  7. 通常用全部大写的变量名表示常量

  8. Python的整数没有大小限制，而某些语言的整数根据其存储长度是有大小限制的，例如Java对32位整数的范围限制在-2147483648-2147483647。Python的浮点数也没有大小限制，但是超出一定范围就直接表示为inf（无限大）。

  9. 在Python中，有三种方式格式化。

    (1) 采用的格式化方式和C语言是一致的，用%运算符格式化字符串

    (2) 使用字符串的format()方法

    (3) 使用以f开头的字符串，它和普通字符串不同之处在于，字符串如果包含{xxx}，就会以对应的变量替换

  10. Python字符串是以Unicode编码的（Java也是）。一般在计算机内存中，统一使用Unicode编码，当需要保存到硬盘或者需要传输的时候，就转换为UTF-8编码，因为后者编码方式更贴合各语言所需字节数，更节省空间，在网络传输和存储时一般用UTF-8。

  11. 当你的源代码中包含中文的时候，在保存源代码时，就需要务必指定保存为UTF-8编码。当Python解释器读取源代码时，为了让它按UTF-8编码读取，我们通常在文件开头写上这两行。

    第一行注释是为了告诉Linux/OS X系统，这是一个Python可执行程序，Windows系统会忽略这个注释；第二行注释是为了告诉Python解释器，按照UTF-8编码读取源代码，否则，你在源代码中写的中文输出可能会有乱码。

  12. 数组在这里成为列表。列表有两种，list是可变长的，tuple是不可变长的

    list有append、pop、index方法来动态改变相当于java里的array，tuple没有，相当于java里的数组。但都可以用classmates[1]、classmates[-2]（倒数第2个）获取数。但你当然可以在tuple里加list，这样就可变了

    注意这里的定义，t不是tuple，是1这个数。这是因为括号()既可以表示tuple，又可以表示数学公式中的小括号，计算结果是1。Python在显示只有1个元素的tuple时，也会加一个逗号,，以免你误解成数学计算意义上的括号。

  13. map在这里称为dict字典。dict的key必须是不可变对象。要保证hash的正确性，作为key的对象就不能变。在Python中，字符串、整数等都是不可变的，因此，可以放心地作为key。而list是可变的，就不能作为key

  14. set依然是set，也就是没有value的map

    要创建一个set，需要提供一个list作为输入集合

  15. java里的switch在python里是match，case _表示“任意值”。同时也可以匹配列表， case ['gcc', file1, *files]表示列表第一个字符串是'gcc'，第二个字符串绑定到变量file1，后面的任意个字符串绑定到*files

  16. 循环，都是用for xx in xx的形式，range()用于生成整数序列，range(101)表示生成0-100序列101个数。或者如果想用索引，可用 Python内置的enumerate函数可以把一个list变成索引-元素对，这样就可以在for循环中同时迭代索引和元素本身

​ 可以直接作用于for循环的数据类型有以下几种：

​ 一类是集合数据类型，如list、tuple、dict、set、str等；

​ 一类是generator，包括生成器和带yield的generator function。

​ 这些可以直接作用于for循环的对象统称为可迭代对象：Iterable。

  1. 切片。list、tuple、string都可以用切片操作。

  2. 生成器。g = (x * x for x in range(10))，g是一个generator，保存算法而不是list那样把每个值都保存下来，每次调用next(g)获取当前值，直到计算到最后一个元素，没有更多的元素时，抛出StopIteration的错误。普通函数加上yield能变成生成器，遇到一个yield中断跳出，下一次调用next()将从yield后开始执行。Python的for循环本质上就是通过不断调用next()函数实现的

  3. 类。类的构造方法用__init__函数写，和普通的函数相比，在类中定义的函数只有一点不同，就是第一个参数永远是实例变量self，表示创建的实例本身，init方法把属性绑定到self上，但调用的时候不需要加self，解释器会自己传进去

    私有属性。实例的变量名如果以__开头，就变成了一个私有变量（private），只有内部可以访问，外部不能访问，如上的score。

  4. Python是动态语言，Java是静态语言。动态是说你的数据结构的类型是在运行时确定的，不要求在编译时写死是整数就只能是整数。这也导致了继承体系没那么严格

  5. 获取对象信息。type()返回对应class类型。dir()返回对象的所有属性和方法。

  6. 类似__xxx__的属性和方法在Python中都是有特殊用途的，比如__len__方法返回长度。

  7. 允许多重继承。class Bat(Mammal, Flyable)

  8. 包装set和get，追求简洁的python。@property和@xx.setter

  9. __call__，把实例当作函数来看

  10. 协程。python3.5后使用asyncio库里的事件循环+async/await实现单线程里协程异步。

事件循环是异步编程的核心。它是一个无限循环，负责调度和执行任务。当一个异步任务需要等待 I/O 操作（例如网络请求、文件读写等）时，它会暂停执行并将控制权交回到事件循环。事件循环可以继续执行其他的任务，从而实现并发。

当 I/O 操作完成后，事件循环会恢复该任务的执行。这种模型的优点是可以在单线程中实现并发，避免了多线程编程中的竞态条件和死锁问题。

async关键字修饰函数，相当于说这个函数是协程函数，await只能用在协程函数里，await意思是暂停当前协程，去执行事件循环
